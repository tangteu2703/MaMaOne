# Icons cho Extension

Extension cần các icon sau:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

Bạn có thể:
1. Tạo icon bằng tool online: https://www.favicon-generator.org/
2. Hoặc dùng icon tạm thời từ: https://via.placeholder.com/

Để tạo icon nhanh, bạn có thể dùng:
- Canva
- Figma
- Photoshop
- Hoặc bất kỳ tool design nào

Icon nên có màu xanh lá (#4CAF50) để phù hợp với theme của extension.
